# Ola-DashBoard
OLA Data Analyst Project

Project Overview:
This project involves performing a series of SQL exercises and creating an interactive Power BI dashboard to analyze and visualize the data. The project demonstrates data extraction, transformation, querying, and visualization skills, providing both detailed solutions to SQL exercises and a user-friendly dashboard for insights.

Contents:

SQL Exercise Solutions:
The detailed solutions for all SQL exercises are documented separately and can be accessed through the following PDF files:
• SQL Exercise 1–10 Solutions: [View PDF](https://drive.google.com/file/d/1BLSsf23Hf1yD5u9EcwBKVHcDAez6mJRK/view?usp=drive_link)

Power BI Dashboard:
The interactive Power BI dashboard provides visual insights into the project data and can be accessed through the following link:
• Power BI Dashboard: [Open Dashboard](https://github.com/Karankukadiya/Ola-DashBoard/blob/main/ola%20dashboard.pbix)

Purpose:

-To practice and demonstrate SQL querying and data manipulation skills.
-To create meaningful visualizations and dashboards for data-driven decision-making using Power BI.
-To provide clear documentation for reproducibility and learning.

How to Use:

-Review the SQL exercise solutions to understand the queries and their results.
-Explore the Power BI dashboard to interactively analyze the data and gain insights.

Note:
All resources, including PDFs and dashboards, are linked for easy access and reference.
